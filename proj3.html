Rock, Paper, Scissors!

Previous Next

Team Size: Personal

Project Type: Artificial Intelligence

Development Tools: Visual Studio/WPF

Programming Language: C#

Duration: Fall 2015

Description:

Rock, Paper, Scissors! is a computer AI that learns how to capitalize on players that fall into patterns while playing the game of the same name. The program has a login screen where the user can create a profile or log into an existing password protected profile. The AI utilizes this for long-term data collection between play sessions. Data for each profile is stored at the end of a play session in JSON files in the computer's documents folder. The data for a profile is then loaded in upon a successful login. The data is used to adjust weighted averages for the computer's long-term learning and, when coupled with the short-term learning which only takes into consideration the current session, makes the computer favor certain throws. A log of play sessions is also saved to the user's desktop so the user can see their play history.
