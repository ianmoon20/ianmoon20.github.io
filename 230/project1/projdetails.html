Reasons for Choosing:

I chose this topic for three main reasons, I'll put them into a convienant list:

  • I've spent a ridiculous amount of time on League of Legends
  • I realize that League is hard for a lot of people, especially beginners, to understand
  • It's an area I feel confident in discussing.
Above and Beyond:

I went above and beyond by dabbling into javascript, which allowed me to implement the ability for the user to cycle images. The text descriptions would automatically cycle to describe the currently displayed image as well.

Improvement:

Honestly, I think the responsive design could use a bit of work. They aren't as smooth as I would like them to be.
