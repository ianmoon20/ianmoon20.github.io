Notes

Borrowed CSS:

This site used the bootstrap template. Also, WC3school and stackoverflow were used as a reference for accomplishing various actions, such as overriding and manipulating bootstrap's carousal gallery.

Above and Beyond:

Manipulated bootstrap in various ways, including creating column layouts, auto-transistioning carousal galleries, and changing the style of the navigation bar. Finally, I created an email form which when filled out and submitted sends me an email with the users desired message and information using PHP.

Challenges:

Bootstrap is strangely picky about the navigation bar font style. It forced me to use !important to override. Also, I don't exactly know which projects to include and which projects to not include, so filling the content pages is definitely a challenge.

Improvement:

In regards to what can be improved, more experience with bootstrap is definitely needed. I don't feel like I have a grasp on all that can be accomplished with this template. Other than that, I think my color choice could be a little better but I don't really have an eye for art so I'm unsure.
